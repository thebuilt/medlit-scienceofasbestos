# Google Sites Embed

After GitHub Pages is live (or your custom domain is live), use this iframe in Google Sites:

```html
<iframe
  src="https://medlit.scienceofasbestos.org/"
  title="Science of Asbestos PubMed Dashboard"
  width="100%"
  height="1000"
  style="border:0;"
  loading="lazy"
  referrerpolicy="no-referrer-when-downgrade"
  allowfullscreen>
</iframe>
```

If you are testing before DNS is ready, temporarily use your GitHub Pages URL in `src`.
